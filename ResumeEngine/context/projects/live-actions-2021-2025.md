# Live Actions
## 2021 – 2025
## NBCUniversal

### Summary
Interactive overlay system enabling real-time viewer engagement during live sports events. Architected core service for time-synced overlays on live and VOD content, creating synchronized interactive experiences aligned with broadcast action. Delivered for Olympics programming and other NBCU live sports properties.

### Business Context
NBCU's OTT platforms needed differentiated interactive experiences to enhance viewer engagement during live broadcasts. Live Actions provided a technical foundation for creating synchronized, real-time interactive elements that could be overlay on top of live streaming content, enabling viewer interaction without disrupting the primary broadcast experience.

### Technical Approach
- **Time-Sync Architecture:** Real-time synchronization of overlays with broadcast video
- **Latency Management:** Sub-second synchronization to ensure interactive elements align with live action
- **VOD Integration:** Post-event application of same overlay infrastructure to VOD content for sustained viewer engagement
- **Multi-Platform:** Consistent experience across streaming and broadcast platforms
- **Scalability:** Supported millions of concurrent viewers during live events

### Major Achievements
- **Olympics Integration:** Led development of Live Actions interactions for Olympics programming
- **Time-Sync Delivery:** Ensured time-synced overlays for viewers, maintaining real-time alignment with broadcast action
- **Overlay Core Service:** Architected and collaborated to create overlay core service for live and VOD interactive experiences
- **Live Scale:** Delivered interactive experiences to millions of concurrent viewers during major sporting events
- **VOD Capability:** Extended overlay functionality to VOD content for post-event engagement

### Technologies & Tools
- **Real-Time Architecture:** Live streaming synchronization, low-latency delivery
- **Overlay Systems:** Real-time graphics rendering, time-code based synchronization
- **Infrastructure:** Cloud-based live event delivery, multi-CDN distribution
- **Video Processing:** Live transcoding, broadcast-grade quality maintenance
- **Standards:** HLS, DASH, SCTE timing standards, live broadcast standards

### Impact & Outcomes
- Enhanced viewer engagement during Olympics and major sports events
- Provided NBCU with competitive interactive features vs. traditional broadcast
- Demonstrated feasibility of synchronized interactive overlays on live broadcast
- Set foundation for future interactive sports products

### Related Features
- Part of NBCU's broader interactive content platform alongside Key Plays and Daily Recaps
- Integrated with ad insertion systems for monetization opportunities
