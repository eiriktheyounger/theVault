## Notes for Generation
- **Do not use banned words:** spearheaded, leveraged, revolutionized, best-in-class, thought leader, passionate about, results-driven, dynamic professional, synergy, impactful
- **Match JD keywords:** If the JD mentions specific tech (e.g., Kafka, gRPC, React), surface those from context roles/projects
- **Emphasize system ownership:** Use "independently designed", "operated autonomously", "end-to-end delivery" where authentic
- **Quantify where possible:** 50M users, 200% growth, 60% reduction, sub-100ms latency
- **Business context first:** Problem → solution → outcome. Never just list features
- **Remote work:** Call out 11+ years remote experience if relevant to role
- **Industry firsts (if authentic):** "First all-digital broadcast ops center", "Early prototypes that informed TWCTV architecture"
